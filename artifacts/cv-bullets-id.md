# CV Bullets (Indonesian)

Pilihan bullet point untuk CV, ditulis dalam bahasa Indonesia. Semua bullet
menggunakan data simulasi dan tidak mengklaim pengalaman operasional nyata.

---

## Opsi 1: Singkat dan teknis

Retail Operations Control Tower
Mengembangkan portfolio project berbasis data simulasi untuk memodelkan workflow
campaign dan store allocation pada jaringan retail multi-store, mencakup data
validation, exception handling, SLA monitoring, dashboard KPI, dan weekly
management report menggunakan Python dan Streamlit.

---

## Opsi 2: Menengah, menonjolkan technical depth

Retail Operations Control Tower
Membangun end-to-end data pipeline berbasis Python yang mensimulasikan workflow
campaign execution pada jaringan 100 toko retail: validasi data (9 rules),
planned-vs-actual reconciliation checks, exception detection
(9 kategori dengan severity scoring dan SLA tracking), metrics layer (12 KPI),
serta dashboard interaktif Streamlit dan weekly operations report. Data
sepenuhnya simulasi (deterministic, seed-based). 327 unit dan integration test
lulus.

---

## Opsi 3: Fokus pada analisis dan decision support

Retail Operations Control Tower
Mengembangkan portfolio project berbasis data simulasi yang memodelkan workflow
campaign dan store allocation pada jaringan retail multi-store. Project ini
mencakup: (1) data simulation layer untuk 8 tabel operasional, (2) validation
engine untuk data integrity, (3) exception engine dengan 9 kategori deteksi,
priority scoring, dan SLA aging, (4) metrics layer dengan 12 KPI dan area manager
scorecard, serta (5) dashboard Streamlit dan weekly report untuk mendukung
prioritisasi tindakan operasional. Semua data bersifat simulasi dan tidak
mengklaim pengalaman operasional nyata.

---

## Catatan penggunaan

- Pilih opsi yang paling sesuai dengan konteks lowongan dan panjang CV yang
  diinginkan.
- Opsi 1 cocok untuk CV ringkas (1 baris tambahan).
- Opsi 2 cocok untuk CV teknis dengan ruang untuk detail.
- Opsi 3 cocok untuk posisi yang menekankan analisis dan decision support.
- Semua opsi menggunakan frasa "portfolio project berbasis data simulasi" untuk
  menjaga kejujuran dan tidak mengklaim pengalaman operasional nyata.
- Tidak mengklaim akses ke data perusahaan manapun.
